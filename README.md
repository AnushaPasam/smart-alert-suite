# EduAlert – Announcement Management System

EduAlert is a modern, role-based platform for managing and receiving campus announcements efficiently. It streamlines communication between administrators, announcers, and students.

## Features

- **Role-Based Access**: Dedicated dashboards for Super Admin (Principal), Admins, Announcers, and Students.
- **Smart Filtering**: Categorize and filter announcements by Academic, Events, Sports, and more.
- **Real-time Notifications**: Instant updates across the campus portal.
- **Engagement Analytics**: Insights into announcement reach and student engagement.
- **PWA Ready**: Installable as a progressive web app for mobile and desktop.

## Tech Stack

- **Frontend**: Vite, React, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **PWA**: vite-plugin-pwa

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <your-repo-url>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

4. Build for production:
   ```sh
   npm run build
   ```
